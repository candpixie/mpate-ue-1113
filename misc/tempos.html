Load One Audio File

Load a single file using `librosa.load()`

file_path = "your_audio.wav"
y, sr = librosa.load(file_path)
print(f"Sample rate: {sr}")

Extract One Tempo

Use `librosa.beat.beat_track()` to find tempo

def get_tempo(file_path):
    y, sr = librosa.load(file_path)
    tempo, _ = librosa.beat.beat_track(y=y, sr=sr)
    return tempo

Print One Tempo

Run the function and print the tempo

file_path = "your_audio.wav"
tempo = get_tempo(file_path)
print(f"Tempo: {tempo} BPM")

Setup File Folder

Put all audio files in a folder

Use `os` to list files

directory = "path/to/audio"
files = os.listdir(directory)

Filter Audio Files

Only get `.wav` or `.mp3` files

audio_files = [f for f in files
    if f.endswith(".wav") or f.endswith(".mp3")]
print(audio_files)

Extract Many Tempos

Loop through all audio files

tempos = []
for file in audio_files:
    file_path = os.path.join(directory, file)
    tempo = get_tempo(file_path)
    tempos.append((file, tempo))

Print All Tempos

Show filenames and their tempos

for file, tempo in tempos:
    print(f"{file}: {tempo} BPM")

Handle Errors

Use `try-except` to avoid errors

for file in audio_files:
    try:
        tempo = get_tempo(os.path.join(directory, file))
        tempos.append((file, tempo))
    except:
        print(f"Error processing {file}")

Sort Tempo List

Sort tempos from slowest to fastest

tempos.sort(key=lambda x: x[1])
print("Sorted Tempos:", tempos)

Find Min & Max

Print slowest and fastest tempos

print(f"Slowest Tempo: {tempos[0]} BPM")
print(f"Fastest Tempo: {tempos[-1]} BPM")
